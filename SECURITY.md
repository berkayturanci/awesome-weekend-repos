# Security Policy

This is a static content site — a generated README and a client-side-only
gallery page with no backend, no user input, and no dependencies. There's
no runtime attack surface, but if you spot something (e.g. a listed repo
turned out to be malicious, or a link points somewhere it shouldn't),
[open an issue](https://github.com/berkayturanci/awesome-weekend-repos/issues/new)
and it'll be addressed promptly.
